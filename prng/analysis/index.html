$ echo -n foo | shasum -a1
0beec7b5ea3f0fdbc95d0dd47f3c5bc275da8a33  -
Note the echo command's -n option is needed to suppress trailing newline character. The following command doesn't produce the hash you'd expect (or desire). The xxd command confirms the different in echo output. 
$ echo foo | shasum -a1
f1d2d2f924e986ac86fdf7b36c94bcdf32beec15  -
$ echo foo | xxd
0000000: 666f 6f0a                                foo.
$ echo -n foo | xxd
0000000: 666f 6f                                  foo
